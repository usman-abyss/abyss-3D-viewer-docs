# Data Format Supported

## GLB data format

Files with .glb.
It can contain information about

1. 3D models,
2. scenes,
3. models,
4. lighting,
5. materials,
6. node hierarchy
7. animations
8. etc.

## Point cloud

Files with .bin

## Octree (Innovated by Abyss)

It contains a group of files: one JSON file and many .bin files.
